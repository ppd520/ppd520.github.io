python内建函数

2020-01-13

截止到3.8.1版本，python一共有69个内建函数

abs(x)

  • 返回x的绝对值
  • 参数类型：整数、浮点、复数
  • 重载 abs, 需要定义 __abs__ 方法

all(iterable)

  • 判断可迭代参数中的元素是否都为真,或者可迭代参数中元素为空
  • 参数类型：可迭代的参数

any(iterable)

  • 判断可迭代参数中的任意元素为真,或者可迭代参数元素为空
  • 参数类型：可迭代的参数

chr(i)

  • 返回unicode码点为i的字符串
  • 参数类型： 整数 0 <= i <= 0x10ffff

  • 与 ord(c) 是一对

  • 参数超出范围，报ValueError错误

  • 为什么参数值 0 <=i <= 0x10ffff （下边是unicode的编码范围）

ord(c)

  • 返回字符所对应的unicode码点
  • 参数类型： 字符 (不是字符串)
  • 与 chr(i) 是一对

repr(object)

  • 返回对象的可打印字符串； 部分对象，eval可以根据可打印字符串还原成对象。

  • 参数类型： 对象 每个对象都有repr方法

  • 重载repr，需要定义__repr__方法

ascii(object)

  • 和repr(object)一样，返回对象的可打印字符串，但是非ascii以\x, \u, \U的形式表示；部分对象，eval可以根据可打印字符串还原成对象。

  • 参数类型： 对象

  • ascii会将非ascii以unicode转译序列表示

bin(x)

  • 返回整数类型的二进制表示，默认以0b开始。

  • 参数类型：整数， 非整数对象 （需要自己实现index,此方法必须返回整数，否则bin会报错）

  • 0b是否需要，可以用如下方法决定

hex(x)

  • 和bin(x)类似

oct(x)

  • 和bin(x)，hex(x)类似

format(value, [format_spec])

  • format_spec 默认是空字符串,效果和str(value)一样

  • format(value, [format_spec]) 被转为 type(value).__format__(value, [format_spec])

  • format_spec 必须为字符串，否则报TypeError

  • format_spec 标准语法：

  • 常见用法

eval(expression [, globals [, locals]])

  • 参数是一个字符串表达式，和可选的globals，和locals
  • globals 全局命名空间，是一个字典
  • locals 局部命名空间，是一个mapping对象
  • 模块级别， globals和locals是一样的

exec(object [, globals [, locals]])

  • 参数是字符串或者代码对象
  • 返回值是None

filter(function, iterable)

  • 如果function是None，则认为所有为false的元素都移除了，即不做任何变化

breakpoint(*args, **kws)

callable(object)

  • 返回True，如果object可调用，否则返回False；
  • 如果实现了call()方法则可调用；

compile(source, filename, mode, flags=0, dont_inherit=False, optimize=-1)

  • source：字符串或者AST对象
  • filename： 代码文件名称
  • mode： exec，eval，single
  • flags： 变量作用域
  • flags和dont_inheritSHI YO你过来控制编译器的编译参数，类似gcc的编译参数

setattr(object, name, value)

  • object: 对象/类
  • name: 字符串，对象属性
  • value: 属性值

getattr(object, name [, default ])

  • object: 对象/类
  • name: 字符串， 对象属性

hasattr(object, name)

  • object: 对象/类
  • name: 字符串， 对象属性

delattr(object, name)

  • object: 对象/类
  • name: 字符串， 对象属性

dir([object])

  • object: 对象
  • 如果没有参数时，返回locals的属性

vars([object])

  • 不带参数，和locals()一样，
  • 返回对象的dict属性

divmod(a, b)

  • a, b: 数字

enumerate(iterable, start=0)

hash(object)

  • 数字相等， 具有同样的hash， 即使类型不同

help([object])

  • positional-only parameters 位置类型的参数，

id(obbject)

input([prompt])

isinstance(object, classinfo)

  • 判断一个对象是否是一个已知类型
  • isinstance考虑继承关系， type不考虑继承关系

issubclass(class, classinfo)

  • 判断一个类是否是classinfo的子类

iter(object [, sentinel])

  • 如果不提供第二个参数， obbject必须实现iter()或者getitem()方法
  • 如果提供了第二个参数，object必须是个可调用的对象(如函数)，iter则创建一个迭代器，每次调用迭代器的next()方法时都会调用object

next(iterator[, default])

  • default, 没有下一个元素时返回的默认值， 否则会抛StopIteration异常

len(s)

  • 返回对象的长度
  • 参数可以是sequence（string, bytes, tuple, list, or range）;
  • 也可以是collection （dictionary, set, or frozen set）

map(function, iterable, …)

  • 返回迭代器
  • iterable多个时，取最小原则

min(iterable, *[,key, default]

min(arg1, arg2, *args[, key])

max(iterable, *[,key,default])

max(arg1, arg2, *args[, key])

  • key, 执行完函数再比较

open(file, mode=’r’, buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)

  • mode: ‘r’,’w’,’x’,’a’,’b’,’t’,’+’
  • error: ‘strict’, ‘ignore’, ‘replace’, ‘surrogateescape’,’xmlcharrefreplace’, ‘backslashreplace’, ‘namereplace’
  • newline: None, ‘’, ‘\r’, ‘\n’, ‘\r\n’
  • bbuffering:

pow(base, exp[, mod])

  • base的exp次方，取模mod

print(*objects, sep=’ ‘, end=’\n’, file=sys.stdout, flush=False)

  • print不能对binary对象操作, 需用file.write()

reversed(seq)

  • seq必须实现reversed()方法
  • 或者len()和getitem()方法

sorted(iterable, *, key=None, reverse=False)

  • 执行完key的结果再排序

round(number[, ndigits])

sum(iterable, /, start=0)

super([type[, object-or-type]])

  • 如何实现多继承协同任务
  • 组合不友好的类
  • mro路径中必须保证方法的存在
  • **kwargs 实现参数匹配
